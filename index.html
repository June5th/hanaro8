사용 흐름

영상 → 퀴즈 → 모의투자/결과

한 줄 요약: 짧게 배우고, 바로 확인하고, 결과로 이해합니다.

1) 영상

5분 영상으로 오늘의 금융 개념을 쉽고 빠르게 이해합니다.

2) 퀴즈

퀴즈로 이해도를 확인하고, 즉시 피드백으로 기억을 강화합니다.

3) 모의투자/결과

선택과 결과를 경험하며 리스크·보상을 자연스럽게 이해합니다.

핵심 기능

빠르게 이해하고, 바로 경험합니다

기능 → 화면 예시로 단계적으로 연결해, 10초 안에 “누가/무엇을/어떻게”가 보이게 만들었습니다.

홈에서 오늘의 추천 학습과 진도를 한눈에 확인합니다.

홈

이 화면에서 무엇을/왜/어떻게

  • 오늘 할 콘텐츠를 바로 추천
  • 학습·퀴즈·투자 진행률을 한 화면에서
  • 처음 사용자도 길을 잃지 않게 안내

영상

핵심만 빠르게 배우고 다음 단계로 연결

  • 5~7분 마이크로 러닝으로 부담 없이 시작
  • 예시 중심으로 개념을 빠르게 이해
  • 바로 퀴즈로 이어져 학습이 고정

퀴즈

즉시 피드백으로 이해도를 확인

  • 정답/오답을 바로 확인
  • 틀린 개념은 핵심만 다시 복습
  • 결과가 다음 추천에 반영

투자

모의투자로 의사결정을 ‘경험’

  • 실제 돈이 아닌 모의 자산으로 안전하게
  • 상황 기반 선택으로 사고력 강화
  • 리스크·보상을 결과로 체감

결과

‘왜’가 이해되게 정리하고 다음 학습으로

  • 변화를 요약해 한눈에 확인
  • 핵심 지표를 쉬운 언어로 설명
  • 개인화 추천으로 반복 학습
